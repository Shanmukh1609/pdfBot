# LangChain_Tutorial

PDF BOt
 Establish the environment
  ``` (shell)
  python -m venv venv
  ```
 Activate the environment
   Ubuntu
  ``` (shell)
  source venv/bin/activate
  ```
   Windows
   ``` (shell)
  .\venv\Scripts\Activate.ps1
  ```
 Download the packages
  ``` (shell)
  pip install -r requirements.txt
  ```
 Run the files
 ``` (shell)
  streamlit run app.py
 ```
